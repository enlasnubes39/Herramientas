🛠️ Herramientas Especiales (ECAM / Zoom)

📑 Reconstructor de PDF

Captura, reconstruye y libera PDFs protegidos de plataformas tipo ECAM. Genera una versión lista para imprimir.

Marcador
javascript:(async function(){ const viewer = window.PDFViewerApplication; if (!viewer) { alert("Abre el PDF primero."); return; } const totalPages = viewer.pagesCount; viewer.pdfViewer.currentScaleValue = "2.5"; const win = window.open("", "_blank"); win.document.write('Captura PDF
Cargando...
'); const container = win.document.getElementById('container'); const status = win.document.getElementById('status'); for (let i = 1; i <= totalPages; i++) { status.innerText = `Procesando Página ${i} de ${totalPages}`; viewer.page = i; await new Promise(r => setTimeout(r, 2200)); const canvas = viewer.pdfViewer.getPageView(i - 1).canvas; if (canvas) { const img = win.document.createElement('img'); img.src = canvas.toDataURL('image/jpeg', 0.95); img.className = 'page-img'; container.appendChild(img); } else { i--; await new Promise(r => setTimeout(r, 1000)); } } status.innerHTML = "✅ LISTO! "; win.focus(); })();

🎥 Zoom Video Hunter

Marcador / Consola
javascript:(function(){ var v = document.querySelector('video'); if (v && v.src) { prompt("¡Enlace cazado! Cópialo:", v.src); } else { alert("Reproduce el video primero."); } })();
🎬 Control de Reproducción Pro

⚡ Super Velocidad

Aumenta la velocidad del video a 2.5x o la que prefieras, superando los límites del reproductor.

Consola
document.querySelector('video').playbackRate = 2.5
Marcador
javascript:(function(){document.querySelector('video').playbackRate=prompt('Velocidad:',2.5);})();

⏩ Salto de Precisión (+10s)

Salta exactamente 10 segundos hacia adelante. Útil para saltar intros o anuncios rápidamente.

Marcador / Consola
javascript:(function(){document.querySelector('video').currentTime+=10;})();

📸 Captura de Diapositiva

Extrae una imagen del momento exacto del video en alta calidad. Se abre en una pestaña nueva.

Marcador / Consola
javascript:(function(){var c=document.createElement('canvas');var v=document.querySelector('video');c.width=v.videoWidth;c.height=v.videoHeight;c.getContext('2d').drawImage(v,0,0);window.open(c.toDataURL());})();

📺 Forzar Fullscreen

Obliga al video a ponerse en pantalla completa aunque el botón de la web no funcione.

Marcador / Consola
javascript:(function(){document.querySelector('video').requestFullscreen();})();
